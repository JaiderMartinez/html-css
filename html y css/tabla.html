NAVEGADORES: CARACTERISTICAS, VENTAJAS Y DESVENTAJAS

NAVEGADOR CARACTERISTICA VENTAJAS DESVENTAJAS
Google Chrome
 Interfaz sencilla y funcional, eficacia, pestañas independientes, soporta mejoras y actualizaciones, navegacion segura, marcadores instantaneos, barra de busqueda, descargas Rápido y fiable, sincronización de datos, buenas opciones para la expansión de la función No hay buena protección de datos, alta carga en la memoria principal cuando hay muchas pestañas abiertas
Mozilla Firefox
 Apto para equipos de sobremesa, portátiles y dispositivos móviles, buena seguridad y excelente protección de datos, el navegador es altamente personalizable, fácil manejo y sincronización de datos y contraseñas Muchas extensiones, alta protección de datos, buena gestión de las contraseñas Ocasionalmente un alto uso de recursos, mayor riesgo de caída
Internet Explorer
Navegador, es un complemento indispensable, era el lider en internet Facilita la comunicación, ofrece información en tiempo real, facilita las compras en otros países a través de internet, permite crear y descargar softwares libres Presenta fallas como la traba de las páginas, la página se carga completa y no por partes, haciendo más lento el proceso, el sistema de carga de las páginas es tardado debido a que utiliza la barra de carga, posee un bajo nivel de seguridad a nivel informático en relación a otros navegadores
Safari
 Sincroniza todas tus contraseñas, marcadores, historial, pestañas y más de forma integrada en tu Mac, iPad, iPhone y Apple Watch Buen rendimiento y fiable para los dispositivos Apple, muy buena protección de datos No es compatible con Windows o Android, falta seguridad ya que las actualizaciones se llevan a cabo muy poco a poco, menos extensiones que otros navegadores
Maxthon
Cuenta con una interfaz de exploración por pestañas, cuentas con personalización de gestos con el ratón, cuentas con las funciones arrastrar y soltar, tienes mayor protección y privacidad, este navegador es compatible con la barra de Google, Live y Yahoo, tienes una barra adicional de utilidades externas. Es simple y muy fácil de usar, es un navegador de alta velocidad, puedes editar y personalizar lo que quieras dentro de el. puedes navegar por distintas páginas web, bloquea los anuncios de los sitios web. Este navegador es limitado siempre y cuando quieras conseguir algo en específico, el navegador no es perfecto como todos los navegadores siempre tienen algo que tienen que llegar a mejorar
Netscape Navigator
Se integra CSS, JavaScript y soporte para los estándares HTML para mantenerse al día con las tendencias de diseño de páginas web Integración con otros programas, múltiples ventanas en pestañas, marcadores Bloqueo de versiones antiguas, no existen nuevas actualizaciones
Opera
 Apto para equipos de sobremesa, portátiles y dispositivos móviles, disponible para Windows, macOS, Linux y Android, borrado automático de la memoria caché, modo turbo para conexiones de Internet lentas, una cómoda barra lateral muestra el Messenger y los marcadores Ampliable con addons, muy seguro, buena sincronización de los dispositivos de escritorio y móviles No está preinstalado, debe descargarse e instalarse
Avant
 es un navegador rápido, estable, de atractivo aspecto visual y que permite tener varias ventanas abiertas dentro de la misma instancia del navegador consiguiendo así ahorrar espacio en la barra de tareas. Además está disponible en 40 idiomas. Sin anuncios, sin Malware, multi-Proceso, navegador Web con menor uso de memoria, descarga ficheros de video, acelerador de descargas integrado, modo vista partida El programa por ser una versión beta, presenta bugs, que causa el bloqueo total de la página, consume muchos recursos de la conexión, es un colador de pasta para atrapar todo tipo de bichos informáticos
